Contractor agrees to provide web development services to Client for the 
following project:

PROJECT DESCRIPTION:
Development of a custom business website with e-commerce functionality, 
including product catalog, shopping cart, payment processing, and admin 
dashboard.

DELIVERABLES:
1. Responsive website design and development (React frontend)
2. Backend API development (Node.js, Express, PostgreSQL)
3. Payment integration (Stripe API)
4. Admin dashboard for product and order management
5. SEO optimization
6. Deployment on AWS cloud infrastructure
7. Source code and documentation
8. 2 hours of training for Client's staff
